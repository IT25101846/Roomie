# Roomie - Hotel Room Reservation System

Spring Boot MVC | Java I/O | JSP | SLIIT Year 1 Semester 2

## Quick Start
1. Open project in IntelliJ IDEA (import pom.xml)
2. Run RoomieApplication.java
3. Open http://localhost:8080

## Default Logins
- Admin: admin / admin123
- Customer: alice@email.com / pass123

## Run Tests
mvn test
